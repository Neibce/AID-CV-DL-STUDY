## Chapter 2. 이미지 처리 기초

### Question1. 그레이 스케일 이미지를 2차원 배열로 나타낸 예시가 궁금합니다.

- **Answer1**: 5x5 이미지가 있다고 가정했을 때, 값이 0이면 검은색, 값이 255면 흰색입니다.

```plaintext
[
  [  0,  50, 100, 150, 200],
  [ 50, 100, 150, 200, 250],
  [100, 150, 200, 250, 255],
  [150, 200, 250, 255, 200],
  [200, 250, 255, 200, 150]
]
```
